Vivado2022.1发布

April 28, 2022 631

导言

Vivado2022.1在4月26号更新到了官网，自从ML版本更新以来，最近几个版本确实没有引入比较大的更新，本次更新也是一样，主要更新了一些器件支持，去年的Aritx7 U+这次增加了支持，我比较感兴趣的点就是，为IP提供实时的资源估计数据，尽管是大版本的更新，但谈不上是什么重大更新（至少对我来说是这样的）。

2022.1 的新增功能：

  • Versal QoR 提升
    • 速度提升 5-8%，具体取决于默认或探索策略
  • 基于 ML 的资源估计
    • 为 IP 提供实时资源估计数据
  • ML Strategy Runs 现可用于 Versal 器件
    • 在迭代难以满足时序的设计时很有用
  • EA 功能
    • 对 Versal 器件的抽象 Shell 支持
  • 在 Vivado ML 企业版和标准版中启用的器件
    • Artix® UltraScale+™ 器件：XCAU15P 和 XCAU10P
    • 其它 Versal® Prime、Premium、AI Core 和 AI Edge 系列器件

这个描述来自官网，还有细致的描述，感兴趣的可以自己官网去看。
